== Description ==

your regiter users can upload the file you choose. 

== Installation ==

You need to create a directory at the root of your site and put it in chmod 777

You install the plugin

To configure it :

- widget appearance
- add it to your sidebar
- in "way", you put the name of the directory that you have created
- in extensions, the type of file that you authorize, just separated by a comma (for example
: RAR,TXT,DOC will allow your users to upload files .rar, .txt, .doc

Warning : It is possible that you should need to change your PHP configuration
